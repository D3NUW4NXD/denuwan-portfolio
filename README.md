# Denuwan Portfolio

A modern cyberpunk-themed personal portfolio website for **D.M. Chathura Denuwan**, built using HTML, CSS, and JavaScript.

## About

This portfolio represents my work as a Software Engineering undergraduate, developer, designer, and digital creator from Sri Lanka.

The website includes:

- Animated cyberpunk interface
- Dark neon purple theme
- Faded cyber city background
- Loading animation
- Glitch hero text
- Education section
- Selected projects section
- Skills section
- Contact section
- AI assistant style chat widget
- Responsive layout for desktop and mobile

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Responsive Web Design
- CSS Animations

## How to Use

1. Download or clone this repository.

```bash
git clone https://github.com/D3NUW4NXD/denuwan-portfolio.git
```

2. Open the project folder.

```bash
cd denuwan-portfolio
```

3. Open `index.html` in your browser.

## How to Customize

### Change Profile Image

Replace this file:

```text
assets/denuwan.jpg
```

### Change Logo

Replace this file:

```text
assets/logo.png
```

### Change Contact Email

In `index.html`, find:

```text
your-email@example.com
```

Replace it with your real email address.

### Change Resume

Replace this file:

```text
assets/Chathura-Denuwan-CV.pdf
```

## GitHub Pages Deployment

1. Go to your GitHub repository.
2. Open **Settings**.
3. Go to **Pages**.
4. Select:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

5. Click **Save**.

Your website will be live after a few minutes.
